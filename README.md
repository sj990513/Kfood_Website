- **서버 :** AWS EC2, ubuntu

  
- **프레임워크 :** Spring Boot 3.0.6

  
- **추가 프레임워크/라이브러리 :** Spring Security, Tymleaf, Jdbc Template

- **IDE :** IntelliJ IDEA 2023, My SQL WorkBench
